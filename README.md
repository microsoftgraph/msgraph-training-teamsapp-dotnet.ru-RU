# <a name="microsoft-graph-training-module---build-microsoft-teams-apps-with-the-microsoft-graph-sdk"></a>Учебный Graph Microsoft — Microsoft Teams приложения с помощью SDK Graph Microsoft

Этот модуль познакомит вас с работой с SDK microsoft Graph для доступа к данным в Microsoft 365 путем создания приложения Microsoft Teams с помощью ASP.NET Core.

## <a name="lab---build-microsoft-teams-apps-with-the-microsoft-graph-sdk"></a>Lab — создание Microsoft Teams приложений с помощью Graph SDK

В этой лаборатории вы создайте консольное приложение с помощью Библиотеки проверки подлинности Майкрософт (MSAL) для доступа к данным в Office 365 с помощью microsoft Graph.

- [Microsoft Teams Руководство Graph Microsoft](https://docs.microsoft.com/graph/tutorials/teams)
